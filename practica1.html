INTRODUCCIÓN A LAS APLICACIONES WEB DE ESCRITORIO

¿Qué son las aplicaciones web de escritorio?

Una aplicación de web de escritorio es un programa que se instala en el ordenador y tiene una interfaz de aplicación pero utiliza las funciones de la web.


¿Qué tipo podemos encontrar?

  • Red social --> Facebook
  • Reproducción audiovisual --> Youtube
  • Compras online --> Wallapop
  • Reproductor de música --> Spotify
  • Chatear y llamar --> WhatsApp

¿Qué ventajas tienen las aplicaciones web de escritorio sobre las aplicaciones web a través del navegador?

Ventajas

  • Tiempo de respuesta mas rapido
  • Facilita el uso de teclas en cliente
  • Es más robusta
  • Se puede hacer todo lo que permita el software

Desventajas

  • Requiere instalación en cada cliente
  • Se tiene que actualizar en cada cliente
  • Normalmente se hace para un sistema operativo específico

Cita al menos 3 ejemplos distintos y sus funcionalidades

  • Facebook: Por medio de esta red, puedes encontrar personas conocidas, interactuar con ellas, participar en grupos que discuten temas de tu interés, compartir contenido (imágenes, texto, vídeo), enviar y recibir mensajes, hacer contactos, realizar búsquedas, hacer anuncios, etc.
  • Pinterest: Pinterest es una red social basada en contenidos visuales (imágenes y vídeos). Una de sus principales características es la proactividad de los usuarios, ya que la red está pensada para inspirarles y motivarles a la hora de llevar a cabo sus proyectos de estilo de vida.
  • Telegram: Con Telegram, puedes enviar mensajes, fotos, videos y archivos de cualquier tipo (doc, zip, mp3, etc.), como también crear grupos de hasta 200.000 personas o canales para hacer difusiones a audiencias ilimitadas. Puedes escribir a tus contactos del teléfono y encontrar personas a través de sus nombres de usuario.
